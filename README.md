# Messaging App (Front)

## Messaging App (Front) - Overview

This is the eight project of the "NodeJS" course within "The Odin Project".

This project is used to practice creating fullstack application.

In particular, this project aims to create a messaging application.

*Note: This repository is for the client. This side of the application only includes the app's interface.*

For the API backend visit [this](https://github.com/Dur4nt3/messaging_project_back) repository.

## Skills Demonstrated

- React

- React Router

- RESTful API

- NodeJS

- Express

- PassportJS (JWT)

- PostgreSQL

- Prisma ORM

- supertest (via Jest)
